# FrameBoard React Example

Vite + React example for `@frameboard/react`.

## Run

From the repository root:

```bash
pnpm dev:react
```

Or from this folder:

```bash
pnpm dev
```

## Demonstrates

- registering multiple screens
- loading, empty, populated, permission, and processing states
- browser FrameBoard canvas
- app-shell wrapper
- package CSS import
- screenshot export controls

# FrameBoard React Native / Expo Example

Expo web example for `@frameboard/react-native`.

## Run

From the repository root:

```bash
pnpm dev:native
```

Or from this folder:

```bash
pnpm start
```

## Demonstrates

- Expo integration
- React Native screen fixtures
- `useResponsiveDimensions()`
- compact vs regular layout behavior
- app shell rendering with bottom tabs
- board mode and focus mode
